# IAya Agency Website V2.0 (Staging)

## Architecture
- **Framework:** React + Vite + TypeScript
- **Styling:** Tailwind CSS with Custom OKLCH palette
- **Design:** Bento Grid 2.0, Glassmorphism

## Folder Structure
- `src/components/ui`: Reuseable components (Glassmorphism & Bento).
- `src/hooks`: Custom hooks (Language detection, scroll animations).
- `src/services`: API interactions (Directus, Google Analytics).
- `src/data`: Local constants and fallbacks.

## Setup
1. `npm install`
2. `npm run dev`

## Colors
- Background: `oklch(15% 0.02 260)`
- Orange: `oklch(65% 0.20 45)`
- Turquoise: `oklch(70% 0.15 190)`
